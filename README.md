# com.uextension.bootstrap

## About

The `com.uextension.bootstrap` package allow services initialization in specific order as preloaded assets.

## Development

Clone the repository in a unity project at the location `Packages/com.uextension.bootstrap`.

## Release

- Update the `CHANGELOG.md` to describe the latest changes of the new version
- Merge your work to the `master` branch
- Tag the merge commit with the new version number `major.minor.revision`

The CI/CD pipeline will automatically build the package and make it available.
